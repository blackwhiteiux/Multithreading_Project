# Многопоточная обработка данных (DataProcessor)

Проект демонстрирует применение многопоточности в Java для параллельной обработки данных. Система принимает наборы чисел, распределяет их вычисление по пулу потоков и собирает результаты в потокобезопасную коллекцию.

## Функции
*   Асинхронная обработка множества задач в фиксированном пуле потоков (10 потоков)
*   Потокобезопасное хранение результатов с использованием `ConcurrentHashMap`
*   Мониторинг активных задач через атомарный счётчик (`AtomicInteger`)
*   Элегантное завершение работы пула потоков (graceful shutdown)
*   Поиск результатов по имени задачи с возвратом `Optional`

## Структура проекта
*   `CalculateSumTask` — класс задачи, реализующий `Callable<Integer>`. Принимает список чисел и вычисляет их сумму. Имитирует длительную операцию через `Thread.sleep()`.
*   `DataProcessor` — основной управляющий класс, который:
    *   Принимает задачи на выполнение (`submitTask`)
    *   Запускает их в пуле потоков (`executeAll`)
    *   Отслеживает прогресс выполнения
    *   Собирает и хранит результаты
    *   Предоставляет доступ к результатам
*   `Main` — демонстрационный класс, создающий 100 случайных задач и запускающий обработку.
